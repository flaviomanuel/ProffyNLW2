> ## How to use
```bash 
# Clone Repository
$ git clone https://github.com/flaviomanuel/proffy
```

* ## API Project

```bash 
$ cd proffy/server

$ yarn install

$ $ yarn start
```

API URL  http://localhost:3333/

* ## Web Project

```bash 
$ cd proffy/web

$ yarn install

$ $ yarn start
```

Results http://localhost:3000/

* ## Mobile Project

```bash 
$ cd proffy/mobile

$ yarn install

$ $ yarn start
```

Install expo app in your smartphone.
